## Getting Started

First, create .env.local:
```bash
touch .env.local
```
Paste the environment variables

GATSBY_API_ENDPOINT <br />
GATSBY_API_KEY <br />
GATSBY_ACCOUNT_ID='007fc49c-ca8a-438c-9caf-85299a8ef6a1' <br />

Second, run the development server:

```bash
yarn next dev
```
